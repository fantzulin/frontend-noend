## 在這個滿地 AI 的時代，只要一行字就能寫出 code 的世界，我還能踏上前端工程師的旅程嗎...
最近在找工作、準備作品、準備 GitHub、準備履歷，突然想起自己以前會整理寫 code 的過程，回過頭來讀這篇文章
<a href="https://fantzulin.notion.site/Front-End-Developer-Training-874f86cebb2d4c289d465b94e0a1018c" target="_blank">
Front End Developer Training
</a>
，當初期許自己能成為前端工程師的心情記錄，現在的我，還是一樣嗎？

離開上一間公司後，我開始了自由業的生活，像是在打支線任務，偶爾看到沒遇過的小怪就去敲兩下，打完好像獲得了一些什麼，但是不打其實也不會怎麼樣，不過中途有遇到幾個隱藏關卡，導致我的技能樹開始變化，路上走走停停到了現在，其實我還是不知道自己是不是一名前端工程師，大概是因為我還沒有在一間公司有正式的前端工程師職稱吧！

我最近在想，在這個滿地 AI 的時代，只要一行字就能寫出 code 的世界，我還能踏上前端工程師的旅程嗎？如果 AI 能寫出前端 90% 的 code，那我還需要多少能力值才能當前端工程師？路上的副本職缺永遠都是要有經驗的冒險家，那我的副本入場券什麼時候才會被我打到呢？

2024 年底和 David 一起破了隱藏關卡，開啟了我的 Web3 地圖，我也從中找到了一點樂趣，我想我對這個遊戲世界似乎還沒完全死心，我想重新記錄這段旅程的心情，我有機會破關嗎？又或者說這個冒險有終點嗎？

期待我二轉成前端工程師的那天，還有穿上 Web3 裝備的那天，到那時候我再來看這篇文章。

<blockquote>
前端工程師可以是畫面的魔法師，也可以是發射資料的弓箭手，還能是詠唱體感的服事。
如果可以，我也想當前端工程師。
</blockquote>
